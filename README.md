# URCine

omdbapi에서 제공한 API로 영화 추천 및 검색 사이트를 구현해 보았습니다.
영화 검색과 영화 정보만 제공하는 API로 표현의 한계가 있어 특정 아이디를 수집하여
검색어를 넣지 않아도 특정 리스트가 생성되도록 구현하였습니다.
